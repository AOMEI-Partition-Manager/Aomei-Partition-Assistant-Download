## Installation & Setup Guide for AOMEI Partition Assistant

#### System Compatibility

- **Supported Windows versions:** Windows 7, 8, 10, 11 (32-bit and 64-bit)
- **Architecture support:** Compatible with both x86 and x64 systems
- **Additional Requirements:** Administrator privileges for disk operations
- **Disk space required:** Approximately 100-150 MB for the application

#### Installation Methods

**Direct Installation:**

1. Download the AOMEI Partition Assistant installer package for Windows.
2. Run the downloaded `.exe` installer file with administrator privileges.
3. Follow the on-screen setup wizard to complete the installation.
4. Launch AOMEI Partition Assistant from the Start Menu or desktop shortcut.

#### First-Time Setup & Configuration

1. **Launch Application:** On first launch, the disk partition software displays all connected drives and partitions.
2. **System Compatibility Check:** The partition manager for windows verifies disk structures and partition layouts.
3. **Pending Operations:** All modifications are queued and applied after confirmation, preventing accidental data loss.

#### Core Partition Operations

1. **Resize/Move Partition:**
   - Select the partition to resize from the disk map.
   - Drag partition boundaries or enter exact sizes for resizing.
   - Apply changes after previewing the new partition layout.

2. **Merge Partitions:**
   - Combine adjacent partitions into a single larger partition.
   - Preserve data from both partitions during the merge process.
   - Useful for consolidating disk space in this partition management tool.

3. **Split Partition:**
   - Divide a large partition into two separate partitions.
   - Allocate specific sizes for each new partition.
   - Maintain data integrity during the split operation.

#### Key Features & Advanced Options

- **Disk Cloning:** Clone entire hard drives or specific partitions for migration or backup using this disk cloning utility.
- **OS Migration:** Move Windows operating system to SSD or larger drive without reinstalling.
- **Partition Recovery:** Recover lost or deleted partitions from damaged drives.
- **MBR/GPT Conversion:** Convert between Master Boot Record and GUID Partition Table formats.
- **SSD Alignment:** Optimize partition alignment for SSD performance and longevity.

#### Troubleshooting

- **Operation Not Allowed:** Some system partitions may be locked during Windows operation. Use bootable media option.
- **Partition Cannot Resize:** Check for unmovable system files. Use defragmentation before resizing.
- **Cloning Fails:** Ensure destination drive has sufficient capacity. Check for bad sectors on source drive.
- **Data Loss Prevention:** Always backup critical data before major partition operations.

#### Updates & Maintenance

- Manual updates require downloading the latest installer from the official website.
- The partition manager supports creating bootable media for offline operations.
